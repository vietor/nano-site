# nano-site

基于 Next.js 的轻量级内容管理系统，支持文章发布、媒体管理和 Markdown 编辑。

## 功能

- 文章 CRUD，Markdown 编辑与预览
- 媒体文件上传与管理，支持类型筛选
- 基于 Cookie 的用户认证
- 模板化页面渲染
- SQLite 存储，配置管理

## 技术栈

Next.js 16 / React 19 / TypeScript / Tailwind CSS 4 / SQLite

## 快速开始

```bash
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000)。
